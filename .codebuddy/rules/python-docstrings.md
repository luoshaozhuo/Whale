# Python 文档字符串规则

- 模块、类、公共函数必须提供文档字符串。
- 文档字符串风格统一采用 Google 风格。
- 文档字符串应简洁说明职责、输入、输出和必要的异常。
- 当函数无参数或返回值时，不强制编写 `Args` 或 `Returns` 小节。
- 文档字符串应解释接口语义与约束，不要简单复述代码表面含义。
- 新增或修改公共接口时，应同步更新文档字符串。
- 文档字符串规范以 `pydocstyle` 与 `flake8-docstrings` 的检查结果为准。
